# Image Size Guide for Service Icons

## Recommended Image Sizes for SEO Content Writing Service

### 1. Home Page (Services.tsx)
- **Display Size**: 56px × 56px (w-14 h-14)
- **Recommended Image Size**: **224px × 224px** (for retina/2x displays)
- **Minimum Size**: 112px × 112px
- **Container**: 96px × 96px (w-24 h-24)
- **Format**: PNG with transparent background
- **Aspect Ratio**: 1:1 (square)

### 2. Services Details Page (ServicesDetails.tsx)
- **Display Size**: 40px × 40px (w-10 h-10)
- **Recommended Image Size**: **160px × 160px** (for retina/2x displays)
- **Minimum Size**: 80px × 80px
- **Container**: 64px × 64px (w-16 h-16)
- **Format**: PNG with transparent background
- **Aspect Ratio**: 1:1 (square)

### 3. Service Image Header (ServiceImage component)
- **Container Height**: 128px (h-32)
- **Full Width**: Responsive
- **Recommended Image Size**: **512px × 512px** (for high quality)
- **Minimum Size**: 256px × 256px
- **Format**: PNG with transparent background
- **Aspect Ratio**: 1:1 (square) or 4:3

## Best Practices

1. **Use 2x resolution** for retina displays (double the display size)
2. **PNG format** with transparent background for best quality
3. **Optimize images** to reduce file size (use tools like TinyPNG)
4. **Square aspect ratio** (1:1) works best
5. **File size**: Keep under 100KB for fast loading

## File Naming Convention

Suggested naming:
- `seo-content-writing-icon.png` (224x224px for home)
- `seo-content-writing-icon-small.png` (160x160px for details)
- `seo-content-writing-hero.png` (512x512px for header)

## Where to Place Images

Place images in: `/public/images/services/`

Example:
- `/public/images/services/seo-content-writing-icon.png`
- `/public/images/services/seo-content-writing-icon-small.png`
- `/public/images/services/seo-content-writing-hero.png`
